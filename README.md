# toutiao

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### 项目目录

**※components:组件配置目录**

**※router/index.js:路由配置位置，可以配置地址**

**※views:页面目录，写完需要在router/index.js:配置**

**App.vue:SPA主要页面，由App.vue显示所有需要看的内容**

**main.js:对App.vue进行vue的配置**
